== Description ==

This plugin allows you to easily scrap images and links from other websites simply by entering the URL
See http://anthony.strangebutfunny.net/my-plugins/image-and-link-scraper/ for help
 
== Installation ==

1. Upload `the directory "alex-image-and-link-scraper"` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Image Scraper" in your settings menu, then place a URL in the URL field and choose links or images and click submit.
